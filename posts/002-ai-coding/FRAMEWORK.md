# 培训文档框架

**标题**：AI 工具实战：用 AI 提升你的工作效率
**时长**：60 分钟 | **受众**：会写 SQL，没用过 AI 编程工具

---

## 第一部分：AI 正在改变我们的工作方式（5 分钟）

- AI 能帮你提高工作效率——无论是重复的还是复杂的工作
- 今天的路线图：工具认知 → 技能扩展 → 实战效果

---

## 第二部分：AI 工具全景（20 分钟）

- 对话类：Claude.ai / ChatGPT
  - 适合：写报告、分析数据、解释 SQL、整理信息
- 知识管理类：NotebookLM
  - 适合：上传文档提问、生成摘要、知识库搭建
- 创作类：Google AI Studio
  - 适合：生成代码框架、原型搭建、多模态内容生成
- AI IDE 类：Claude Code / Cursor / GitHub Copilot
  - 适合：AI 辅助写代码、自动化脚本、数据处理

---

## 第三部分：Agent Skills（10 分钟）

- 什么是 Agent Skills：给 AI 安装"专业技能包"的开放标准
- 生态现状：Anthropic 发布，OpenAI / Cursor / GitHub 已采用，96,000+ skills 可用
- Live Demo：
  - Claude Code 安装 xlsx skill + statistical-analysis skill
  - 分析宏观经济 GDP.xlsx → Before / After 对比

---

## 第四部分：AI Coding 实战（15 分钟）

- 什么是 AI Coding，什么是 Vibe Coding（顺带解释，30 秒）
- 和 SQL 的类比：你已经在做类似的事
- Live Demo：宏观经济 GDP 分析 Web App 完整制作过程
  - Step 1 → NotebookLM：收集宏观经济解读文章，生成结构化内容
  - Step 2 → Google AI Studio：生成幻灯片演示系统的 React Web 框架
  - Step 3 → Claude Code + react-best-practice skill：下载到本地，用 Agent Skill 迭代项目
  - Step 4 → 展示最终成品
- 衔接台词：*"你们刚才看到的 NotebookLM、AI IDE、Agent Skills，在这个项目里全都用到了"*

---

## 第五部分：下一步（10 分钟）

- 三个行动清单
  - 今天：打开 Claude.ai，用 AI 辅助写下一份报告
  - 本周：试用 NotebookLM，上传一份常用参考文档
  - 进阶：下载 Claude Code，装一个 skill 试试
